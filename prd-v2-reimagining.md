# Plan: Re-imagining the Vision (PRD v2.0)

## Objective
To transition the portfolio from a functional placeholder to an avant-garde "Digital Installation" by redefining the PRD with a focus on "Post-AI Authenticity" and "Engineered Soul."

## Key Changes
1.  **Aesthetic Shift**: Move from standard UI patterns to "Tactile Brutalism" and "High-End Editorial."
2.  **Interaction Model**: Implement "Weighted Motion" and "Kinetic Typography."
3.  **Content Presentation**: Transition to an "Asymmetric Bento Grid" that treats different mediums (poetry vs. code) with unique visual textures.
4.  **Technical Depth**: Add a "Machine Mode" to showcase underlying logic and MX (Machine Experience) optimization.

## Implementation Phases

### Phase 1: The "Engineered" Foundation
- Define the new CSS variable system for "Tactile Brutalism" (sharp 0px corners, 1px solid borders).
- Establish the 28px "Global Grid" rhythm as a visible background element.
- Select and implement a variable font system for headlines (Kinetic Typography).

### Phase 2: The "Soulful" Interaction
- Refactor the Hero section into a viewport-scaled typographic installation.
- Implement the "Asymmetric Flow" for the About section with delayed, weighted fades.
- Add "Human Fingerprint" layers (CSS noise, grain overlays).

### Phase 3: The "Living Shelf" Grid
- Rebuild the Portfolio as a non-linear Bento Grid.
- Apply "Category Textures" (Serif for Poetry, Mono for Code).
- Enhance the Archive detail pages into "Editorial Spreads."

## Verification & QA
- **Visual Audit**: Compare against "Anti-Generic" manifesto.
- **Performance Check**: Maintain 95+ Lighthouse scores despite high-fidelity motion.
- **Accessibility**: Ensure high-contrast brutalism meets WCAG AAA.
