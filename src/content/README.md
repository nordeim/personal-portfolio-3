# Content Folders

Use these folders to update the website content.

- Put your main profile photo in `portrait/`.
- Put slide-specific hero photos in `portrait/creative-technologist/`, `portrait/project-archive/`, and `portrait/open-to-collaborate/`. The `project-archive` folder powers the visible Growing Collections slide.
- Put portfolio collection items in the matching `collections/` folder.
- Put an image with the same filename next to a collection text file to use it as that item's preview image.
- Files named `PUT_...` are helper notes only and will not appear on the website.

Example:

`collections/poetry/morning-note.md`
`collections/poetry/morning-note.jpg`

The site reads these files during local development and when you build the site.
