# Profile Photo Folders

This folder supports a fallback portrait and three slide-specific portraits.

## Fallback Portrait

Put a general portrait directly in this folder. It will be used whenever a slide-specific folder is empty.

Recommended fallback filename:

`nicholas-portrait.jpg`

## Slide-Specific Portraits

Use these subfolders when you want each hero slide to show a different image:

- `creative-technologist/`
- `project-archive/`
- `open-to-collaborate/`

Supported image types:

`.jpg`, `.jpeg`, `.png`, `.webp`, `.avif`
